Apa itu Database?

Database adalah kumpulan data terstruktur yang disimpan secara elektronik. Ada dua jenis utama: Relational (RDBMS) dan NoSQL.

RDBMS menyimpan data dalam tabel (baris dan kolom) yang saling terhubung. Menggunakan SQL (Structured Query Language). Cocok untuk data terstruktur, transaksi keuangan, dan integritas data tinggi.

NoSQL menyimpan data tanpa skema tetap (dokumen, grafik, key-value). Cocok untuk big data, real-time apps, data tidak terstruktur.

Menggunakan MySQL dengan XAMPP

XAMPP adalah paket server lokal yang berisi Apache, MySQL, PHP, dan Perl. Untuk mulai menggunakan MySQL:

  1. Download dan install XAMPP dari apachefriends.org.
  2. Jalankan XAMPP Control Panel, start Apache dan MySQL.
  3. Buka browser, akses http://localhost/phpmyadmin untuk mengelola database via phpMyAdmin (antarmuka web).
  4. Atau gunakan command line: buka terminal, arahkan ke folder MySQL (misal C:\xampp\mysql\bin) dan jalankan mysql -u root.
  5. Buat database baru: CREATE DATABASE namadb; lalu gunakan: USE namadb;
  6. Jalankan perintah SQL seperti di simulasi untuk membuat tabel dan mengelola data.
