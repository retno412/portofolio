Apa itu GitHub?

GitHub adalah platform berbasis web untuk hosting repositori Git. Git sendiri adalah sistem version control yang melacak perubahan kode. Dengan GitHub, tim dapat berkolaborasi, melakukan review kode, dan mengelola proyek secara terpusat.

Manfaat: backup kode, kolaborasi, pelacakan perubahan, branch & merge, pull requests, integrasi dengan tools lain.

Perintah Dasar Git

git init

Membuat repositori baru di folder lokal.

git add .

Menambahkan semua perubahan ke staging area.

git commit -m "pesan"

Menyimpan perubahan dengan pesan.

git status

Menampilkan status file.

git log

Melihat riwayat commit.

git branch

Menampilkan atau membuat branch.

git checkout -b nama

Membuat dan pindah branch baru.

git merge branch

Menggabungkan branch ke branch aktif.

git clone url

Menyalin repositori dari remote.

git push

Mengirim commit ke remote.

git pull

Mengambil perubahan dari remote.

git remote add origin

Menambahkan remote repository.

Cara Penggunaan GitHub

  1. Buat akun di github.com (misal: retno412).
  2. Buat repositori baru (New repository), beri nama, jangan lupa init README.
  3. Clone ke lokal: git clone https://github.com/retno412/nama-repo.git
  4. Lakukan perubahan, add, commit seperti simulasi di atas.
  5. Push ke remote: git push origin main (perlu autentikasi token).
  6. Jika kolaborasi: gunakan git pull sebelum push, buat branch untuk fitur, lalu pull request di GitHub.
