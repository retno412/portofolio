üè∑Ô∏è Kelas & Objek

Kelas adalah blueprint/cetakan. Objek adalah instance dari kelas. Contoh: Kelas Mobil, objek mobilSaya.

class Mobil { ... }
Mobil mobilSaya = new Mobil();

üîí Encapsulation

Menyembunyikan data internal dan hanya menyediakan akses melalui metode. Menggunakan private dan getter/setter.

private int umur;
public int getUmur() { return umur; }

üå≥ Inheritance

Kelas mewarisi atribut dan metode dari kelas induk. Mencegah duplikasi kode. Contoh: MobilListrik extends Mobil.

class MobilListrik extends Mobil { ... }

üîÑ Polymorphism

Metode yang sama dapat memiliki perilaku berbeda di kelas turunan (override) atau parameter berbeda (overload).

void suara() { ... } // di kelas induk
@Override void suara() { ... } // di anak

// Output akan muncul di sini
