Apa itu Mobile Programming?

Mobile programming adalah pengembangan aplikasi untuk perangkat mobile (Android, iOS). Pendekatan cross-platform memungkinkan menulis satu kode untuk dua platform sekaligus, menghemat biaya dan waktu. React Native dan Flutter adalah dua framework populer.

React Native (dari Meta) menggunakan JavaScript/TypeScript. Keunggulan: ekosistem React yang luas, berbagi logika dengan web, dan performa mendekati native.

Cara Memulai React Native

  1. Install Node.js dan npm/yarn.
  2. Install Expo CLI atau React Native CLI.
  3. Untuk Expo: npm install -g expo-cli, lalu expo init my-app.
  4. Untuk React Native CLI: ikuti petunjuk di reactnative.dev.
  5. Masuk ke folder: cd my-app, lalu expo start atau npx react-native run-android.
  6. Struktur dasar: App.js adalah komponen utama. Gunakan View, Text, dll.
