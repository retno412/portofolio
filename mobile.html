Cross-platform mobile dengan Dart & Google

Apa itu Mobile Programming?

Mobile programming adalah pengembangan aplikasi untuk perangkat mobile (Android, iOS). Pendekatan cross-platform memungkinkan menulis satu kode untuk dua platform sekaligus, menghemat biaya dan waktu. Flutter dan React Native adalah dua framework populer.

Flutter (dari Google) menggunakan bahasa Dart. Keunggulan: rendering UI sangat cepat dan konsisten (pixel perfect), hot reload, dan widget yang kaya.

Cara Memulai Flutter

  1. Install Flutter SDK dari flutter.dev.
  2. Install editor (Android Studio, VS Code) dan plugin Flutter/Dart.
  3. Jalankan flutter doctor untuk cek kesiapan.
  4. Buat project baru: flutter create my_app.
  5. Masuk ke folder: cd my_app, lalu flutter run (bisa pilih emulator atau device).
  6. Struktur dasar: lib/main.dart adalah pintu masuk. Gunakan MaterialApp dan Scaffold.
