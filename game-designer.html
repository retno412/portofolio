Game Designer

Sang "Sutradara" Game

Menentukan aturan main, level, cerita, dan mekanik game agar seru. Game Designer membuat Game Design Document (GDD), merancang keseimbangan, dan memastikan pengalaman pemain sesuai visi.

Tools yang digunakan:

  • Microsoft Word / Google Docs (untuk GDD)
  • Miro / Lucidchart (diagram alur)
  • Unity / Unreal (untuk prototyping cepat)
  • Trello / Jira (manajemen proyek)
