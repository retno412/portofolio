React Lengkap

React adalah library JavaScript untuk membangun antarmuka pengguna (UI). Dikembangkan oleh Facebook. React berbasis komponen, menggunakan JSX, dan virtual DOM untuk performa tinggi.

  • JSX â€“ sintaks seperti HTML di dalam JavaScript.
  • Props â€“ data yang dikirim dari parent ke child (read-only).
  • State â€“ data internal komponen yang bisa berubah.
  • Virtual DOM â€“ representasi ringan DOM untuk update efisien.

ðŸ”¹ Komponen Fungsional dan JSX

ðŸ”¹ Props (Properties)

Props adalah argumen yang diteruskan ke komponen. Bersifat read-only.

function Card({ judul, children }) { return ( <div className="card"> <h3>{judul}</h3> <div>{children}</div> </div> ); }

ðŸ”¹ State dan useState

State digunakan untuk data yang berubah. Hook useState mengelola state dalam komponen fungsional.

import React, { useState } from 'react'; function Counter() { const [count, setCount] = useState(0); return ( <div> <p>Klik: {count} kali</p> <button onClick={() => setCount(count + 1)}> Tambah </button> </div> ); }
Klik: 0 kali

ðŸ”¹ useEffect (Side Effects)

Menangani efek samping seperti fetch data, subscription, atau manipulasi DOM.

import React, { useState, useEffect } from 'react'; function DataFetcher() { const [data, setData] = useState(null); useEffect(() => { fetch('https://api.example.com/data') .then(res => res.json()) .then(setData); }, []); // array kosong = hanya sekali saat mount return <div>{JSON.stringify(data)}</div>; }
