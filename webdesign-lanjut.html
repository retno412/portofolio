Framework CSS (Bootstrap/Tailwind)

Bootstrap: Framework berbasis komponen. Siap pakai dengan desain konsisten. Menggunakan kelas seperti btn btn-primary.
Tailwind: Framework utility-first. Membangun desain dengan kelas utilitas seperti bg-blue-500 p-4. Fleksibel dan ringan.

Animasi & Transisi

Transisi: perubahan halus antara dua keadaan (misal hover). Properti: transition: all 0.3s.
Animasi @keyframes: animasi kompleks dengan banyak tahap.

Preprocessor (SASS)

SASS memperluas CSS dengan variabel, nesting, mixin, dan fungsi. Ditulis dalam file .scss, dikompilasi menjadi CSS biasa.

$primary: #2F8F9D;
.btn {
  background: $primary;
  &:hover {
    opacity: 0.8;
  }
}

SEO Dasar

Search Engine Optimization: praktik agar situs mudah ditemukan mesin pencari. Elemen penting: meta tags (title, description), heading hierarchy, alt text pada gambar, semantic HTML, kecepatan halaman.

<title>... <meta name="description"> alt="gambar"

<!-- Menggunakan utility classes (seperti Tailwind) -->
<style>
/* SASS syntax (dikompilasi) */
$primary: #2F8F9D;
.btn { background: $primary; transition: transform 0.3s; }
.btn:hover { transform: scale(1.05); }
@keyframes fadeIn { from { opacity:0; } to { opacity:1; } }
.hero { animation: fadeIn 2s; }
</style>
