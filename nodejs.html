Node.js adalah runtime JavaScript di server. Memungkinkan JavaScript berjalan di backend, dengan event-driven, non-blocking I/O.

Contoh server sederhana:

const http = require('http');
http.createServer((req,res) => res.end('Halo')).listen(3000);
