CSS3 Lengkap

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mendesain tampilan halaman web. CSS3 adalah versi terbaru dengan fitur canggih seperti animasi, flexbox, grid, dan media queries.

ðŸ”¹ Cara Menyisipkan CSS

  • Inline: menggunakan atribut style pada elemen HTML.
  • Internal: dalam tag <style> di bagian <head>.
  • Eksternal: file .css terpisah yang di-link dengan <link>.
/* Contoh CSS eksternal */ body { background-color: #f0f3fa; } h1 { color: #1E2F4F; font-size: 2rem; }

ðŸ”¹ Selector Dasar

element .class #id * (universal) element, element element element (descendant) element > element (child) element + element (adjacent)
/* Contoh selector */ p { color: blue; } /* semua elemen p */ .card { border: 1px solid; } /* class */ #header { background: black; } /* id */ div p { margin: 10px; } /* p di dalam div */

ðŸ”¹ Box Model

Setiap elemen HTML direpresentasikan sebagai kotak yang terdiri dari: margin, border, padding, dan content.

.box { width: 200px; padding: 20px; border: 5px solid #2F8F9D; margin: 15px; }

ðŸ”¹ Layout: Flexbox

Flexbox digunakan untuk layout satu dimensi (baris atau kolom).

.container { display: flex; justify-content: center; /* rata tengah horizontal */ align-items: center; /* rata tengah vertikal */ gap: 10px; } .item { flex: 1; }
