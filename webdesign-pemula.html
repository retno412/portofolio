Elemen HTML yang memiliki makna (seperti <header>, <nav>, <article>, <footer>) membantu mesin pencari dan pembaca layar memahami struktur halaman. Ini meningkatkan SEO dan aksesibilitas.

CSS Dasar (Selector & Box Model)

Selector digunakan untuk menarget elemen HTML (misal: element, class, id). Box Model menjelaskan bahwa setiap elemen terdiri dari margin, border, padding, dan content. Ini penting untuk layout.

Selector | Box Model

Teks, Warna, Gambar

Mengatur teks dengan properti seperti color, font-size, font-family. Warna bisa menggunakan nama, hex, rgb. Gambar ditambahkan dengan tag <img> dan diatur dengan CSS.

Teks | Warna | Gambar

Struktur Halaman Sederhana

Kombinasi HTML dan CSS untuk membuat layout dasar: header, navigasi, konten, footer. Menggunakan elemen semantik dan CSS untuk tata letak sederhana.

Contoh Layout

Demo: Halaman Web Sederhana (Hasil Langkah-langkah)

<!DOCTYPE html>
<html>
<head>
  <title>Demo Pemula</title>
  <style>
    body { font-family: Arial; background: #f0f3fa; }
    header { background: #1E2F4F; color: white; padding: 20px; text-align: center; }
    nav { background: #2F8F9D; padding: 10px; }
    nav a { color: white; margin: 0 15px; }
    .content { padding: 20px; }
    .box { background: #f7d44b; padding: 10px; width: 200px; margin: auto; border-radius: 10px; }
    footer { background: #1E2F4F; color: white; text-align: center; padding: 10px; }
  </style>
</head>
<body>
  <header><h1>Website Saya</h1></header>
  <nav><a href="#">Home</a> <a href="#">Tentang</a></nav>
  <div class="content">
    <h2>Judul Artikel</h2>
    <p>Ini adalah contoh teks dengan warna biru tua.</p>
    <img src="https://via.placeholder.com/150" alt="Contoh gambar">
    <div class="box">Kotak dengan padding</div>
  </div>
  <footer>© 2025</footer>
</body>
</html>
