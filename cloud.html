‚òÅÔ∏è Apa itu Cloud Computing?

Cloud computing adalah penyediaan sumber daya komputasi (server, penyimpanan, database, perangkat lunak) melalui internet. Model layanan utama: IaaS (infrastruktur), PaaS (platform), SaaS (perangkat lunak). Developer dapat fokus pada kode tanpa mengelola infrastruktur fisik.

Penyedia Cloud Utama

AWS

Layanan terbanyak, ekosistem luas. Populer: EC2, S3, Lambda, RDS.

Belajar AWS

Google Cloud

Unggul di data analytics, Kubernetes (GKE). Populer: Compute Engine, Cloud Functions, Firestore.

Belajar GCP

Microsoft Azure

Integrasi dengan produk Microsoft. Populer: Virtual Machines, Functions, Cosmos DB.

Belajar Azure
