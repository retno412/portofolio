JavaScript ES6+ Lengkap

JavaScript adalah bahasa pemrograman yang membuat halaman web interaktif. ES6 (ECMAScript 2015) memperkenalkan banyak fitur modern yang sekarang menjadi standar.

ðŸ”¹ Cara Menyisipkan JavaScript

  • Inline: menggunakan atribut onclick, dll.
  • Internal: dalam tag <script> di HTML.
  • Eksternal: file .js terpisah yang di-link dengan <script src="...">.
<script> console.log('Hello World'); </script> <script src="script.js"></script>

ðŸ”¹ Variabel: var, let, const

var let const
let nama = 'Budi'; // bisa diubah const umur = 20; // tidak bisa diubah var lama = 'hindari'; // function-scoped

ðŸ”¹ Tipe Data

String, Number, Boolean, Object, Array, Null, Undefined.

let teks = "Hello"; let angka = 100; let benar = true; let siswa = { nama: 'Ani', umur: 17 }; let daftar = [1,2,3]; let kosong = null; let tidakAda = undefined;

ðŸ”¹ Operator dan Percabangan

// Operator: + - * / % === !== < > || && ! if (angka > 50) { console.log('besar'); } else { console.log('kecil'); } // Switch switch (nilai) { case 1: break; default: break; }

ðŸ”¹ Looping

for (let i=0; i<5; i++) { } while (kondisi) { } array.forEach(item => { }); for (let item of array) { }

ðŸ”¹ Fungsi (Function)

// Deklarasi fungsi function tambah(a,b) { return a + b; } // Fungsi anonim (expression) const kali = function(a,b) { return a*b; }; // Arrow function (ES6) const kurang = (a,b) => a - b; // Parameter default const sapa = (nama = 'Tamu') => `Halo ${nama}`; // Rest parameter const jumlah = (...angka) => angka.reduce((a,b) => a+b);

ðŸ”¹ Object dan Array (ES6+)

// Destructuring object const siswa = { nama: 'Budi', umur: 20 }; const { nama, umur } = siswa; // Destructuring array const warna = ['merah', 'hijau']; const [pertama, kedua] = warna; // Spread operator const arr1 = [1,2]; const arr2 = [...arr1, 3,4]; const obj1 = { a:1, b:2 }; const obj2 = { ...obj1, c:3 }; // Template literals console.log(`Nama: ${nama}, umur: ${umur}`);

ðŸ”¹ DOM Manipulation

Mengakses dan memanipulasi elemen HTML.

// Memilih elemen const judul = document.getElementById('judul'); const divs = document.querySelectorAll('.card'); // Mengubah konten judul.textContent = 'Judul Baru'; judul.innerHTML = '<span>Hello</span>'; // Event listener button.addEventListener('click', function() { alert('Diklik!'); });

Klik tombol untuk mengubah teks ini.

ðŸ”¹ Promise dan Async/Await

Menangani operasi asinkron seperti fetch API.

// Promise fetch('https://api.example.com') .then(response => response.json()) .then(data => console.log(data)) .catch(error => console.error(error)); // Async/Await (ES8) async function ambilData() { try { const res = await fetch('https://api.example.com'); const data = await res.json(); console.log(data); } catch (error) { console.error(error); } }

ðŸ”¹ Module (import/export)

// file math.js export const PI = 3.14; export function tambah(a,b) { return a+b; } // file main.js import { PI, tambah } from './math.js';
