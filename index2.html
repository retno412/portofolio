💻 Bab 2: Belajar Teknis – Urutan Efektif

Ikuti urutan ini agar tidak bingung. Setiap bulan fokus pada satu area, lengkap dengan materi, simulasi, dan proyek nyata.

📘 2.1 Bulan 1: HTML & CSS (Dasar Web)

HTML Semantik

<header>, <nav>, <main>, <article>, <section>, <footer> — membuat struktur yang bermakna.

Simulasi: 
<header>
            <h1>Website Saya</h1>
        </header>
        <nav>
            <a href="#">Beranda</a>
        </nav>
        <main>
            <article>...</article>
        </main>

Form & Tabel

<form>, <input>, <select>, <table> — untuk input data dan menampilkan data tabular.

Simulasi form: 
<form>
            <label>Nama:</label>
            <input type="text">
            <button>Kirim</button>
        </form>

CSS Dasar

Warna (color, background), font (font-family, font-size), margin, padding, border.

Simulasi: 
body {
            background-color: #f0f0f0;
            font-family: Arial, sans-serif;
        }
        h1 {
            color: blue;
            margin-top: 20px;
        }

Flexbox & Grid

Layout modern: display: flex dan display: grid untuk mengatur posisi elemen.

Flexbox contoh: 
.container {
            display: flex;
            justify-content: center;
            gap: 10px;
        }
✅ Proyek Bulan 1: Buat halaman profil pribadi (tentang kamu, hobi, foto) dan clone halaman login sederhana (seperti punya Google). Gunakan HTML & CSS saja.
Sumber gratis: W3Schools, freeCodeCamp, YouTube: Web Programming UNPAS.

⚡ 2.2 Bulan 2: JavaScript (Dasar & DOM)

Variabel & Tipe Data

let, const, string, number, boolean, array, object.

let nama = "Budi";
        const umur = 20;
        let isStudent = true;
        let hobi = ["membaca", "coding"];

Fungsi & Conditional

function, if/else, switch.

function sapa(nama) {
            if (nama) {
                return "Halo, " + nama;
            } else {
                return "Halo!";
            }
        }

Loop

for, while, forEach untuk perulangan.

for (let i = 0; i < 5; i++) {
            console.log("Iterasi " + i);
        }

DOM Manipulation & Event

Mengubah elemen HTML, menangani klik, input, dll.

document.getElementById("tombol")
            .addEventListener("click", function() {
                alert("Tombol diklik!");
            });
✅ Proyek Bulan 2: To-do list interaktif (tambah, hapus tugas) dan kalkulator sederhana (penjumlahan, pengurangan, dll).
Sumber: W3Schools JavaScript, freeCodeCamp, YouTube: Web Programming UNPAS.

🔧 2.3 Bulan 3: Git & GitHub + Responsive Design

Git Dasar

git init, add, commit, branch, merge. Version control untuk melacak perubahan kode.

git init
        git add .
        git commit -m "first commit"
        git branch -M main
        git remote add origin https://...
        git push -u origin main

GitHub

Hosting repository, kolaborasi, membaca kode orang lain.

Buat repo di GitHub, clone, push, pull.

Responsive Design

Media queries, viewport, unit relatif (%, vw, vh).

@media (max-width: 600px) {
            body { font-size: 14px; }
        }

Hosting Gratis

Netlify / Vercel: deploy dari GitHub dengan mudah.

1. Push project ke GitHub.
2. Hubungkan ke Netlify.
3. Deploy otomatis.

✅ Proyek Bulan 3: Portfolio website responsive dengan 3-4 halaman (beranda, tentang, proyek, kontak). Gunakan HTML, CSS, dan JavaScript minimal. Push ke GitHub dan deploy ke Netlify.
Sumber: Learn Git Branching, YouTube: Dea Afrizal (Git & GitHub).

🗄️ 2.4 Bulan 4-5: Backend (Pilih PHP atau Node.js)

PHP (laris di pasar lokal)

  • Server: XAMPP / Laragon (Apache + MySQL).
  • Database: MySQL, phpMyAdmin.
  • CRUD: Create, Read, Update, Delete data.
  • Autentikasi: Login, register, session.
Contoh koneksi database (PHP): 
$conn = new mysqli("localhost", "root", "", "db");
        $sql = "SELECT * FROM users";
        $result = $conn->query($sql);

Node.js (modern, startup)

  • Server: Node.js + Express.
  • Database: MongoDB (NoSQL) atau MySQL.
  • CRUD: REST API dengan Express.
  • Autentikasi: JWT, bcrypt.
Contoh route Express: 
app.get('/users', (req, res) => {
            User.find({}, (err, users) => {
                res.json(users);
            });
        });
✅ Proyek Bulan 4-5: Aplikasi catatan sederhana dengan login (misal: catatan pribadi). Pengguna bisa registrasi, login, tambah, edit, hapus catatan. Gunakan PHP+MySQL atau Node.js+MongoDB.
PHP: Web Programming UNPAS; Node.js: Dea Afrizal.

⚛️ 2.5 Bulan 6: Framework & CMS

React (Frontend)

Komponen, state, props, hooks (useState, useEffect). Membangun UI interaktif.

function Counter() {
                const [count, setCount] = useState(0);
                return <button onClick={() => setCount(count + 1)}>
                    {count}
                </button>;
            }

Vue (Frontend)

Alternatif React, lebih ringan. Directive, component, Vuex.

<template>
                <button @click="count++">{{ count }}</button>
            </template>
            <script>
            export default { data() { return { count: 0 } } }
            </script>

WordPress (CMS)

Membuat website tanpa coding banyak, cocok untuk klien yang ingin mudah dikelola. Pelajari pembuatan tema anak (child theme) dan plugin dasar.

Instal WordPress, pilih tema, gunakan page builder (Elementor).

✅ Proyek Bulan 6 (pilih salah satu):
  • Blog sederhana dengan React + API (misal dari JSONPlaceholder).
  • Company profile dengan WordPress (tema kustom sederhana).
React: Dea Afrizal; Vue: Programmer Zaman Now; WordPress: Web Programming UNPAS.

⏳ Bab 3: Strukturisasi & Jadwal Belajar

3.1 Membuat Jadwal Realistis (Contoh)

  • Senin-Jumat: 1 jam malam (20.00-21.00)
  • Sabtu: 3 jam eksplorasi proyek
  • Minggu: istirahat/review materi

3.2 Metode Belajar Aktif

Jangan hanya nonton tutorial, coding bareng. Buat catatan digital agar mudah diingat.

📝 Simulasi Membuat Catatan di Notion

1

Kunjungi notion.so dan buat akun gratis.

2

Buat halaman baru dengan klik + Add a page.

3

Buat database untuk mencatat materi: gunakan template Table atau List.

4

Tambahkan kolom: Materi, Sumber, Status (belum/selesai), Catatan.

5

Setiap selesai nonton video atau baca artikel, catat poin penting dan kode yang dicoba.

Contoh tampilan Notion (ilustrasi)

💡 Tips: Gunakan template "Learning Roadmap" yang tersedia di Notion atau buat sendiri sesuai gaya belajarmu.

3.3 Mengatasi Rasa Stuck

Teknik "rubber duck debugging": jelaskan kode kamu pada bebek karet (atau teman) – dengan menjelaskan, seringkali kamu menemukan solusinya sendiri.

Jangan malu bertanya! Bergabunglah dengan komunitas developer Indonesia yang aktif:

SantriKoding Discord Telegram Belajar Web Dev Programmer Handal Indo Dev Community

🎯 Tips: Saat bertanya, jelaskan apa yang sudah dicoba, sertakan screenshot atau potongan kode agar lebih mudah dibantu.

🌍 Bab 5: Memahami Dunia Freelance

5.1 Pasar Indonesia

Platform: Sribulancer, Projects.co.id, Fastwork, grup Facebook. Harga awal Rp300k - Rp1jt.

5.2 Pasar Luar Negeri

Upwork, Fiverr, Freelancer.com. Tantangan: bahasa Inggris, pembayaran PayPal/Payoneer. Buat proposal spesifik.

5.3 Perbedaan Ekspektasi

Klien luar: formal, tepat waktu, harga tinggi. Lokal: lebih santai, sering bertemu langsung.

💬 Bab 6: Komunikasi dengan Klien

6.1 Frasa Inggris Penting

✉️ Perkenalan: "Hi [Client], I'm [Nama], a web developer with experience in [skill]. I saw your project about [judul] and I believe I can help."
📝 Menanyakan detail: "Could you provide more details about the design you have in mind?"
📊 Progress: "I've completed the homepage layout. Here's the preview link. Please let me know if there are any revisions."
💰 Menagih: "I've finished the project as agreed. I'll send the invoice via [platform]."

6.2 Template Negosiasi

"I understand your budget. For that price, I can offer [fitur terbatas]. Alternatively, for the full scope, my rate is [harga]."

6.3 Etika Profesional

Konfirmasi spesifikasi, buat kontrak sederhana, jangan ghosting.

💰 Bab 7: Menentukan Harga & Mengelola Keuangan

7.1 Metode Penentuan Harga

Hitung biaya hidup + target tabungan. Tarif per jam: Rp50k-150k (lokal) / $15-$30 (luar). Fixed price = estimasi jam × tarif + buffer 20%.

7.2 Mengelola Uang

Pisahkan rekening bisnis, sisihkan 20-30% untuk pajak & dana darurat. Gunakan BukuWarung atau Money Lover.

📋 Bab 8: Manajemen Proyek & Tools

8.1 Alat Bantu Kerja

Trello/Notion untuk tugas, WhatsApp/Slack untuk komunikasi, Figma untuk desain.

8.2 Menghindari Scope Creep

Buat Statement of Work (lingkup pekerjaan). Jika klien minta fitur baru: "Itu bisa ditambahkan dengan biaya tambahan sebesar [harga]."

8.3 Backup Kode

Gunakan GitHub private, backup database rutin.
