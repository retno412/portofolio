Rute Komunikasi Data

Client HTTP Request Server Database Response

Alur umum aplikasi web: client mengirim request melalui protokol HTTP/TCP, server memproses, mungkin mengambil data dari database, lalu mengembalikan response.

üì± Contoh Aplikasi dengan Komunikasi Data

  • Chatting (WhatsApp, Telegram): Menggunakan socket (TCP) untuk pengiriman pesan real-time, protokol XMPP atau custom.
  • Game Online (Mobile Legends, PUBG): Menggunakan UDP untuk update posisi pemain agar cepat, TCP untuk transaksi.
  • Web Browser: Mengirim HTTP/HTTPS request ke server, menerima HTML, CSS, JS.
  • IoT (Internet of Things): Perangkat sensor mengirim data ke server via MQTT atau HTTP.
