Pemrograman Fungsional
Haskell, Scala, dan fitur JavaScript (map, filter, reduce)

Paradigma fungsional memandang komputasi sebagai evaluasi fungsi matematis dan menghindari perubahan state (immutable). Kode lebih ekspresif, mudah diuji, dan cocok untuk pemrosesan data paralel. Bahasa fungsional murni seperti Haskell, atau multi-paradigma seperti Scala dan JavaScript (dengan fitur fungsional).

Bahasa Fungsional

Haskell

Fungsional murni, lazy evaluation, sistem tipe kuat. Digunakan di riset dan industri (Facebook, GitHub).

Pelajari Haskell

Scala

Menggabungkan OOP dan fungsional, berjalan di JVM. Populer untuk big data (Apache Spark).

Pelajari Scala

JavaScript: Map, Filter, Reduce

ðŸ”¹ map()

Membuat array baru dengan hasil pemanggilan fungsi pada setiap elemen.

const angka = [1,2,3];
const kaliDua = angka.map(x => x * 2);
// hasil: [2,4,6]

ðŸ”¹ filter()

Menyaring elemen yang memenuhi kondisi.

const angka = [1,2,3,4];
const genap = angka.filter(x => x % 2 === 0);
// hasil: [2,4]

ðŸ”¹ reduce()

Mereduksi array menjadi nilai tunggal (akumulasi).

const angka = [1,2,3,4];
const jumlah = angka.reduce((acc, cur) => acc + cur, 0);
// hasil: 10
